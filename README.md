## 📋 1. Portada y Propósito
Este proyecto consiste en el diseño y desarrollo de una **librería nativa de JavaScript funcional (sin frameworks ni componentes visuales)** que resuelve de manera integral el problema de la captura y depuración de datos del lado del cliente. 

La librería centraliza validaciones rutinarias de seguridad (complejidad de contraseñas, formatos de correos), control de cadenas de texto y cómputos de fechas comunes, garantizando la integridad de la información antes de ser procesada por un servidor.

---

## ⚙️ 2. Instalación y Configuración
Para integrar las funciones de la utilería en cualquier documento HTML, basta con descargar el archivo `utileria.js` e importarlo al final del bloque `<body>` mediante una etiqueta de script estándar:

```html
<script src="js/utileria.js"></script>
